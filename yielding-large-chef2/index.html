YLC Hard-Coded Note-Taking App


This page should be installable.
This web app is hard-coded in Chrome as a note-taking app (works in Chrome 91-93 (inclusive)).
If you want a non-hard-coded note-taking app, using manifest fields: note-taking-in-manifest (works in Chrome 93+).
Also for testing: empty-note-taking-in-manifest and invalid-note-taking-in-manifest.
